# Implementation Logic & Programming Tricks

Because the RiSC-16 has only 8 instructions, common tasks are performed using specific logic patterns.

## 1. Emulating Missing Instructions

- **Move**: `add rA, rB, r0` (Copies B into A because r0 is 0).
- **Not**: `nand rA, rB, rB` (NANDing a value with itself is NOT).
- **Sub**: (A - B)
    1. `nand rB, rB, rB` (Flip bits of B)
    2. `addi rB, rB, 1` (Add 1 to get Two's Complement)
    3. `add rA, rC, rB` (Add the negative result)

## 2. Managing the PC

- **Unconditional Jump**: `beq r0, r0, label` (Always branches because 0 equals 0).
- **Function Call**: `jalr r7, r1` (Saves return address in r7 and jumps to r1).

## 3. Creating 16-bit Constants

To load `0xABCD` into `r1`:

```text
lui  r1, 0x2AF  # Top 10 bits (0xABCD >> 6)
addi r1, r1, 0xD # Bottom 6 bits (0xABCD & 0x3F)
```
