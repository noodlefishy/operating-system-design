# RiSC-16 Instruction Set Reference

## Opcodes & Logic

| Opcode | Mnemonic | Format | Operation                                    |
|:-------|:---------|:-------|:---------------------------------------------|
| **0**  | `add`    | RRR    | `RegA = RegB + RegC`                         |
| **1**  | `addi`   | RRI    | `RegA = RegB + SignedImm7`                   |
| **2**  | `nand`   | RRR    | `RegA = ~(RegB & RegC)`                      |
| **3**  | `lui`    | RI     | `RegA = Imm10 << 6` (Clears bottom 6 bits)   |
| **4**  | `lw`     | RRI    | `RegA = Mem[RegB + SignedImm7]`              |
| **5**  | `sw`     | RRI    | `Mem[RegB + SignedImm7] = RegA`              |
| **6**  | `beq`    | RRI    | `if (RegA == RegB) PC = PC + 1 + SignedImm7` |
| **7**  | `jalr`   | RRI    | `RegA = PC + 1; PC = RegB`                   |

## Key Implementation Details

1. **Memory Access**: Both `lw` and `sw` use base+offset addressing.
2. **LUI (Load Upper Immediate)**: Used to build 16-bit constants. It places 10 bits into bits 15-6 of the target
   register.
3. **The Syscall Trap**: If `jalr` is executed with a **non-zero** immediate value (bits 6-0), the hardware should
   trigger a System Call using that immediate as the Vector ID.
