# RiSC-16 Hardware Specifications

## Core Architecture

- **Word Size**: 16-bit.
- **Address Space**: 16-bit word-addressed (Address 1 refers to the second 16-bit word, not the second byte).
- **Registers**: 8 General Purpose Registers (R0 through R7).
    - **R0 Constraint**: Hardwired to `0`. Any write to R0 is discarded; any read returns `0`.
- **Execution Cycle**: Standard Fetch-Decode-Execute.

## Instruction Word Formats

Instructions are exactly one word (16 bits) long, divided into three possible layouts:

| Format       | Bits 15-13 | Bits 12-10 | Bits 9-7         | Bits 6-0                    |
|:-------------|:-----------|:-----------|:-----------------|:----------------------------|
| **RRR-Type** | Opcode     | Reg A      | Reg B            | 4 bits (0) + Reg C (3 bits) |
| **RRI-Type** | Opcode     | Reg A      | Reg B            | Signed Immediate (7 bits)   |
| **RI-Type**  | Opcode     | Reg A      | 10-bit Immediate | (Included in Bits 9-0)      |

- **Signed Immediates**: 7-bit values range from `-64` to `63`.
- **Unsigned Immediates**: 10-bit values range from `0` to `1023`.
