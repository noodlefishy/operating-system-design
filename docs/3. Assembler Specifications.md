# Assembler Directives & Pseudo-Instructions

## Pseudo-Instructions (Macros)

These are translated by the assembler into one or more base instructions:

- **`nop`**: (No Operation) -> `add 0, 0, 0`.
- **`halt`**: (Stop execution) -> `jalr 0, 0` with a non-zero immediate.
- **`lli regA, imm`**: (Load Lower Immediate) -> `addi regA, regA, (imm & 0x3F)`.
- **`movi regA, imm`**: (Move 16-bit Immediate) -> A two-step process:
    1. `lui regA, (imm >> 6)`
    2. `lli regA, (imm & 0x3F)`

## Data Directives

- **`.fill [value]`**: Allocates one word of memory initialized to the given value or label address.
- **`.space [n]`**: Allocates `n` words of memory initialized to `0`.

## Syntax Rules

- **Comments**: Start with `#` and continue to the end of the line.
- **Labels**: Must end with a colon (e.g., `start:`) and must be on the same line as an instruction or directive.
- **Numbers**:
    - Hexadecimal: `0x12`
    - Octal: `012` (Leading zero)
    - Decimal: `12`
